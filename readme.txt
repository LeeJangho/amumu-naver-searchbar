== Description ==
Amumu\'s 플러그인 100개 만들기 - 1

이 프로젝트는 100개의 플러그인을 만들어 배포할때까지 계속 됩니다.

네이버 검색창에 원하는 키워드를 넣어 구성하는 간단한 플러그인 입니다.

[amumu-naver-searchbar keyword=""] 형식으로 사용 하시면 됩니다.



== Installation ==

1. FTP 직접 업로드

/wp-content/plugins/amumu-naver-searchbar

2. 플러그인 검색 후 인스톨
